## 📱 MyScanner - Mobile Document Scanner

**MyScanner** is a powerful mobile application built with TypeScript, Node.js, and Expo that transforms your smartphone into a professional document scanner. It combines real-time QR code recognition with advanced document processing capabilities.

### ✨ Key Features

- **🔲 Auto QR Code Scanning**: Effortless automatic QR code detection and recognition with real-time camera preview
- **📄 Document Scanning**: Capture and automatically flatten documents
- **🎯 Image Enhancement**: Intelligent image sharpening and clarity optimization
- **💾 File Export**: Save processed documents as high-quality image files

### 🛠️ Tech Stack

- **Frontend**: React Native with Expo
- **Language**: TypeScript
- **Backend**: Node.js
- **Camera & Processing**: Native mobile APIs with Expo modules

### 📋 Requirements

- Node.js (v16 or higher)
- Expo CLI
- Mobile device or emulator (iOS/Android)

### 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start the development server
expo start

# Run on iOS or Android
# Press 'i' for iOS simulator or 'a' for Android emulator
```

### 📸 How It Works

1. **QR Code Scanning**: Point your camera at any QR code for automatic detection
2. **Document Capture**: Frame your document within the guide and capture
3. **Auto Processing**: The app automatically flattens the perspective and enhances clarity
4. **Save**: Export your processed document as a PNG or JPEG image file

### 🎨 Use Cases

- Digitize receipts and invoices
- Scan business cards and documents
- Create document archives
- Process forms and worksheets
